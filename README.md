# Vplayer

Native music player

## Tech Stack

### Frontend

* React
* TailwindCSS
* HowlerJS

### Backend

* Electron
* LowDB
* Nedb

## Getting Started

1. `yarn install`
2. `yarn start`

## Notes

1. Please run `yarn lint` before commit.
